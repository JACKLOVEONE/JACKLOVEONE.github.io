JDOJ-1119 Fibonacci sequence

题目链接：https://neooj.com:8082/oldoj/problem.php?id=1119

题目大意：求第x到第y项斐波拉契和 mod10000.x, y <= int

总结

  • 主要是构造矩阵

  • 矩阵元素为前缀和 si 表示 a1 + a2 +…..ai

    ai ai+1 ai+2

    ai+1 0 1 0
    ai+2 0 0 1
    ai+3 -1 0 2

  • 写出对应矩阵
    0 0 -1
    1 0 0
    0 1 2
